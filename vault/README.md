# vault/

Shared Obsidian-style knowledge vault for this marketplace. The single place any skill's output
is filed, indexed, linked, and retrieved. Managed by the **vault-keeper** plugin — don't hand-file
here; ask Claude to "save this to the vault" and let the skill place it, add frontmatter, and wire
it into `index.md`.

```
index.md      root MOC — the front door
MOCs/         one map-of-content per topic
notes/        atomic notes (one idea each)
artifacts/    whole-file outputs (reports, decks, transcripts)
assets/       binaries referenced from notes
```
